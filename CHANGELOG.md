# Changelog

All notable changes to this project will be documented here.

---

## [1.0.0] — 2026-01-17

### Added — Physical Items Browser
- Initial release of the Items Browser.
- DnD5e‑native confirm dialog for adding physical items.
- Category grouping for weapons, armor, tools, gear, consumables, ammo, containers, and misc.
- Embedded column headers (Category / Quantity / Cost) aligned to item rows.
- Actor‑sheet‑style quantity widget with +/– buttons and input support.
- Buy Mode (currency subtraction with validation).
- Free Mode (adds items without cost).
- Total cost calculation with live updates.
- Multi‑item selection and batch creation.
- Theme‑compatible CSS using DnD5e system variables.

### Added — Spell Browser Integration Notes
- Spell Browser now documented as a sibling feature.
- Shared UI/UX philosophy between Spell Browser and Physical Browser.
- Notes on spell filtering, known‑spell tinting, and DnD5e‑native styling.
